Almost ready

This app needs a free Firebase backend to sync everyone live. One-time, ~5 minutes.

  1. Go to console.firebase.google.com and create a project.
  2. Add a Web app (the </> icon) and copy the firebaseConfig values.
  3. Enable Authentication → Email/Password.
  4. Create a Firestore Database (start in production mode).
  5. Paste your config into the top of this file where it says firebaseConfig, then reload.

Full step-by-step is in SETUP.md next to this file, including the security rules to paste.

How ranks work

Your tier comes from a Battle Rating that rewards winning and weights bigger events more heavily:

  • Pickup game win ×1 · RTT win ×3 · GT win ×6
  • A draw is worth half a win; a loss earns no points but still counts as a game played.
  • Rating = weighted win points × a win-quality factor, so how much you win and where you win both matter.
  • Play ${''} games to leave Unranked.
